## [v0.9.0-alpha] — 2026-07-08

First alpha release with web dashboard, ChatAgent, CLI, and full cognitive runtime.

### Added
- **CLI shell** (`a2x shell`), **dashboard** (`a2x dashboard`), **monitor** (`a2x monitor`)
- **WorldGraph bootstrap** — 12 concepts + 10 edges pre-loaded on startup
- **ChatAgent** — Ollama/OpenAI backend, 14 tools, context memory, persistence
- **Web dashboard** — live agent cards, WorldGraph force graph, StateField heatmap, Chat tab
- 71 tests across all crates
