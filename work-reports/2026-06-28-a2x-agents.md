## Key Design Decisions

- **AgentState uses `String` for errors** — avoids `Clone` requirement on `AgentError` (which contains `Box<dyn Error>`)
- **All agents implement the `Agent` trait from a2x-core** — `execute()` accepts raw `Packet` for Phase 0
- **CLI sandboxing** — `SandboxMode` enum with None/CommandFilter/Container/Vm variants
- **Orchestrator uses Arc<Mutex<CcsVm>>** — thread-safe VM for potential async dispatch
- **All agents are Phase 0 stubs** — `execute()` returns empty raw packets; full execution in Phase 2+
