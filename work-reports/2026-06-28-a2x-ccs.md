## Key Design Decisions

- **Borrow-safe VM**: `DecodedInstruction` struct extracts all instruction data into owned values before any mutable `self` operations — avoids Rust borrow conflicts
- **petgraph::StableGraph** for WorldGraph: node indices remain stable after deletions
- **FlatStateField** with named regions: simple Vec<f32> with HashMap index, default 1024 floats
- **Σ∞→Opcode mapping**: Intent/Plan/Data operators mapped to 16 CCS opcodes (Phase 0 partial mapping)
- **All operators are Phase 0 stubs** — full neural implementations in Phase 2+
