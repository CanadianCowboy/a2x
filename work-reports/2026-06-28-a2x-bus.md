## Key Design Decisions

- **Transport uses `&mut self`** — InMemoryTransport actually pushes/drains mailboxes
- **WireMessage has `payload: Vec<u8>`** — bus serializes SigmaPacket::to_string() into payload
- **Discovery returns sorted results** — deterministic ordering (HashMap iteration is non-deterministic)
- **Bus integrates Transport + Discovery + Router** — single coordination point
